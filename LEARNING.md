## Día 1 – Variables y Tipos de Datos

Hoy aprendí los fundamentos de Python relacionados con variables y tipos de datos. Entendí cómo declarar variables y cómo almacenar distintos 
tipos de información como números enteros (`int`), decimales (`float`), texto (`str`) y valores booleanos (`bool`).

Practiqué creando múltiples variables y utilizándolas en diferentes contextos, especialmente con `f-strings`, lo que me permitió imprimir mensajes de forma más 
clara y profesional. También trabajé con operaciones básicas como suma y resta, y comprendí cómo Python maneja los tipos de datos utilizando la función `type()`.

Durante la práctica, reforcé la importancia de escribir código limpio y entendible, así como evitar errores comunes como mezclar tipos incorrectamente o sobrescribir variables sin intención.
En general, logré una base sólida sobre cómo funcionan las variables en Python y cómo utilizarlas correctamente dentro de un programa.

## Ejemplo trabajado
nombre = "Yoskar"

edad = 21

print(f"Hola, me llamo {nombre}")

print(f"En 10 años tendré {edad + 10} años")
